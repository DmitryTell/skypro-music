# Приложение Skypro Music

Многофункциональный проигрыватель аудиотреков

## Available Scripts

### `npm run start`

Запустить проект на локальном сервере

### `npm run test`

Запустить тесты

### `npm run build`

Собрать продакшн

### `npm run eject`

## Время на выполнение:

Запланированное: 16 часов
