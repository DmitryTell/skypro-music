# Приложение Skypro Music

Многофункциональный проигрыватель аудиотреков.

- Есть возможность регистрации и авторизации.
- Есть возможность загрузки плейлистов по жанрам.
- Есть возможность ставить (снимать) лайки трекам.
- Есть возможность переключения темы (светлая и темная).

## Технологии

Typescript, React, Styled Components, Routes, Axios, Redux, Redux Toolkit, Redux Toolkit Query, EsLint, Craco Alias, StyleLint

## Инструкция по сборке и запуску

- Установка зависимостей: `npm install`
- Запуск с локального сервера: `npm run start`
- Продакт-сборка: `npm run build`
- Деплой проекта: `npm run deploy`
